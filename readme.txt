== Description ==

Extends Wordpress Gallery Manager

== Installation ==

1. Upload `webpm-gallery.php` to the `/wp-content/plugins/` directory
   or upload through plugin manager. Search for WebPM Gallery
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Done
